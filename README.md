This is NPM Growler: fetches NPM recent update modules and growls as new activies are coming in.

## INSTALLATION

clone from github.

    $ git clone git://github.com/hideo55/node-npm-growler.git

npm link

    $ npm link node-npm-growler

and run

    $ npm-growler
    $ npm-growler --interval 600000
